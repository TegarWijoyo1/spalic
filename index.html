Deskripsi Alat

Sistem pemadam kebakaran gedung perkuliahan otomatis berbasis Programmable Logic Controller (SPALIC) merupakan sebuah alat yang dirancang khusus untuk membantu mendeteksi dan memadamkan kebakaran secara otomatis pada gedung laboratorium teknik 3. Sistem ini dibuat dengan tujuan untuk meningkatkan keamanan dan keselamatan penghuni gedung laboratorium teknik 3 melalui pencegahan dan pemadam kebakaran yang cepat dan efektif, serta memantau kondisi ruangan di gedung perkuliahan melalui website yang dapat diakses oleh pengguna sehingga penghuni gedung laboratorium teknik 3 tidak perlu bergantung pada penanggulangan kebakaran secara manual. Sistem ini menggunakan Programmable Logic Controller (PLC) tipe omron cp1e sebagai kontrol utama yang berfungsi untuk menerima respons dari hasil pembacaan sensor suhu, sensor asap, dan sensor tekanan dari mikrokontroller sehingga jika terdeteksi tanda-tanda kebakaran pada sebuah gedung perkuliahan, PLC akan mengambil tindakan untuk menghidupkan alarm sebagai tanda bahaya dan menghidupkan pompa untuk memadamkan api melalui sprinkle. Mikrokontroller berfungsi sebagai penghubung antara sensor-sensor dan PLC sekaligus untuk pembacaan kondisi lingkungan gedung laboratorium teknik 3 yang akan ditampilkan pada website sebagai pemantauan kondisi gedung yang dapat diakses oleh pengguna. Alat SPALIC memiliki keunggulan dibandingkan dengan sistem pemadam api yang konvensional yang hanya menggunakan perangkat keras atau elektronik sederhana yang hanya dapat memberikan informasi dasar seperti kondisi alarm saat aktif atau pada saat non-aktif sehingga kurang fleksibel, berbeda dengan SPALIC yang dapat memonitoring kondisi ruangan gedung laboratorium teknik 3 dari jarak jauh melalui website yang dapat diakses oleh pengguna sehingga lebih fleksibel.
